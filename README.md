# Cycle & Pregnancy Tracker for SillyTavern

Продвинутое расширение для фронтенда SillyTavern, которое добавляет систему отслеживания менструального цикла, овуляции и беременности персонажа ({{user}}).

## 🌟 Основные возможности

- **Отслеживание цикла:** Автоматический расчет фаз (Менструация, Фолликулярная, Овуляция, Лютеиновая) в реальном времени.
- **Система зачатия:**
  - Автоматический шанс зачатия при отправке сообщений во время фазы овуляции.
  - Рандомное определение количества детей (один, близнецы или тройня).
  - Рандомное определение пола (Мальчик, Девочка или Смешанный для близнецов).
- **Мониторинг беременности:** Точный счётчик недель с момента зачатия.
- **Интеграция:** Полная поддержка переменных SillyTavern (например, `{{user}}`).
- **Интерфейс:** Красивый, современный и легкий UI, полностью на русском языке.

## 🚀 Установка

1. Откройте **SillyTavern**.
2. Перейдите в меню **Extensions** (Расширения) -> **Install Extension** (Установить расширение).
3. Вставьте ссылку на этот репозиторий:
   `https://github.com/dimitritafatu-ctrl/st-cycle-tracker`
4. Нажмите **Install**.
5. После установки расширение появится в списке настроек под заголовком **"Трекер цикла и беременности"**.

## 🛠 Настройка

1. **Включить трекер:** Активирует работу расширения.
2. **Женские гениталии ({{user}}):** Включает логику цикла и возможности беременности для пользователя.
3. **Авто-зачатие:** Если включено, расширение будет "бросать кубик" на шанс беременности при каждом вашем сообщении в фазу овуляции (12-16 день цикла).
4. **Начать новый цикл:** Кнопка для сброса и установки даты начала последней менструации на "сегодня".
5. **Сбросить беременность:** Позволяет очистить данные о текущей беременности.
